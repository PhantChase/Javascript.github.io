### Strings
```
`Down on the sea`
"Lie on the ocean"

"con" + "cat" + "e" + "nate"
`half of 100 is ${100 / 2}`
```
### Unary Operators
```
console.log(typeof 4.5) // → number
console.log(typeof "x") // → string
```
### Boolean Values
```
console.log(3 > 2) // → true 
console.log(3 < 2) // → false
```
### Empty Values
### Automatic Type Conversion
```
console.log(8 * null) // → 0 
console.log("5" - 1) // → 4 
console.log("5" + 1) // → 51 
console.log("five" * 2) // → NaN
```
